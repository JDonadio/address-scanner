Installation

Clone the source:

git clone git@github.com:javierbitpay/address-scanner.git cd address-scanner

Install http://bower.io if you haven't already:

npm install -g bower

Build address-scanner

bower install

About Address-scanner

This tool implements a simple interface to get your unspent BTC from a copay-wallet backup and allow send that amount to a new wallet.
